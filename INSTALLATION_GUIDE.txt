================================================================================
USING WITH MOBILE (NGROK SETUP)
================================================================================

WHY NGROK?
----------
Mobile browsers require HTTPS to access camera/microphone.
ngrok provides free HTTPS tunnel to your localhost.

SETUP:
------
1. Download ngrok: https://ngrok.com/download
2. Extract ngrok.exe to any folder
3. Open Command Prompt in that folder
4. Run: ngrok http 80
5. You'll see output like:
   
   Forwarding: https://abc123.ngrok-free.app -> http://localhost:80
   
6. Use the HTTPS URL on your mobile phone!
7. Example: https://abc123.ngrok-free.app/Nigiri/

IMPORTANT:
- Keep ngrok running while testing
- URL changes every time you restart ngrok
- Free tier is enough for testing


================================================================================
TROUBLESHOOTING
================================================================================

PROBLEM: "Database connection error"
SOLUTION: 
- Make sure MySQL is running in XAMPP
- Run setup.bat again
- Check db.php has correct credentials (root, no password)

PROBLEM: "Unknown database 'docathome'"
SOLUTION:
- Database wasn't created
- Run setup.bat
- Or manually create database in phpMyAdmin

PROBLEM: "Unknown column 'status'"
SOLUTION:
- Old SQL file imported
- Run setup.bat (it adds missing column)
- Or use docathome_complete.sql instead

PROBLEM: "Permission denied" for camera
SOLUTION:
- Use HTTPS (ngrok)
- Check browser permissions
- Try different browser

PROBLEM: "Connection failed" in video call
SOLUTION:
- Both devices must use same ngrok URL
- Check internet connection
- Refresh both browsers (Ctrl+F5)

PROBLEM: Video call stuck on "Connecting..."
SOLUTION:
- Check browser console (F12) for errors
- Verify TURN servers are reachable
- Disable firewall temporarily to test

PROBLEM: Chat messages not appearing
SOLUTION:
- Check MySQL is running
- Verify messages table exists
- Check browser console for errors
